# STUDYMATE QA Checklist

## 1. 사용자 인증 및 온보딩

### 로그인/회원가입
- [ ] 구글 소셜 로그인 정상 작동
- [ ] 네이버 소셜 로그인 정상 작동
- [ ] 로그인 후 토큰 저장 확인
- [ ] 자동 로그인 기능 작동
- [ ] 로그아웃 정상 작동

### 온보딩 프로세스
- [ ] 기본 정보 입력 (ObInfo)
  - [ ] 이름, 생년월일, 성별 입력
  - [ ] 다음 단계 이동
- [ ] 언어 설정 (ObLang)
  - [ ] 모국어 선택
  - [ ] 학습 언어 선택
  - [ ] 레벨 선택
- [ ] 관심사 선택 (ObInt)
  - [ ] 관심사 카테고리 표시
  - [ ] 다중 선택 가능
- [ ] 파트너 선호도 (ObPartner)
  - [ ] 선호 국가 선택
  - [ ] 선호 성별 선택
- [ ] 스케줄 설정 (ObSchedule)
  - [ ] 요일별 시간 설정
  - [ ] 타임존 설정
